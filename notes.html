The Web

The Web (or World Wide Web) is a collection of documents and other files. Most of the documents are made using HyperText Markup Language (HTML).

What is HTML?

HTML stands for Hypertext Markup Language. It's used to define the structure or layout of a web page.

Markup

Markup gives instructions to the browser on things like how text should be displayed or arranged. It is an indicator of when something has a special meaning.

Optional Closing Tags

Two elments that have optional closing tags are the p and li.
  • The p element does not require a closing tag because when the browser sees another p element or an element that cant be inside of the last one, it will automatically close that p element.
  • The li element is similar in that the browser knows a list cant exist on its own, so it knows when it sees another li to close the prior one.
